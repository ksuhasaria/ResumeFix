# ResumeFix

ResumeFix is a production-ready web application designed to help job seekers in India optimize their resumes for ATS (Applicant Tracking Systems) and land more interviews.

## Features
- **Premium UI**: Modern, mobile-first design with a focus on clarity and speed.
- **ATS Optimization**: AI-driven analysis to identify missing keywords and formatting issues.
- **Professional Rewriting**: Automatically generates a high-impact, role-specific resume.
- **Interview Q&A**: Personalized preparation based on your target role and profile.
- **7-Day Action Plan**: Step-by-step strategy for job applications in the Indian market.
- **PDF Export**: Download your optimized resume as a professional PDF.

## Tech Stack
- **Frontend**: Next.js 15 (App Router), React, Framer Motion, Lucide Icons.
- **Styling**: Vanilla CSS (CSS Modules) with a custom design system.
- **Backend**: Next.js API Routes.
- **Tools**: jsPDF, html2canvas for PDF generation.

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Deployment
The app is designed to be easily deployed on Vercel via GitHub integration.
